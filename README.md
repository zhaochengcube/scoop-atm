# Scoop Bucket for ATM

## Installation

```powershell
scoop bucket add atm https://github.com/cubezhao/scoop-atm
scoop install atm
```

## Upgrade

```powershell
scoop update atm
```

## Uninstall

```powershell
scoop uninstall atm
```

## About

This bucket provides [ATM (AI Tools Manager)](https://github.com/cubezhao/ai-tools-mng) for Windows users.
